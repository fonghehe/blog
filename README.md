## 技术栈

- **框架**: VitePress 1.6.4
- **语言**: Markdown + Vue 3 + TypeScript
- **部署**: GitHub Pages + GitHub Actions
- **包管理**: pnpm
- **主题**: 自定义深色/浅色主题

## 🚀 本地运行

```bash
pnpm install
pnpm docs:dev
```

## 🏗️ 构建

```bash
pnpm docs:build
```

## 👁️ 预览构建结果

```bash
pnpm docs:preview
```
