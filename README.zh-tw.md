## 技術棧

- **框架**: VitePress 1.6.4
- **語言**: Markdown + Vue 3 + TypeScript
- **部署**: GitHub Pages + GitHub Actions
- **套件管理**: pnpm
- **主題**: 自訂深色/淺色主題

## 🚀 本地執行

```bash
pnpm install
pnpm docs:dev
```

## 🏗️ 建置

```bash
pnpm docs:build
```

## 👁️ 預覽建置結果

```bash
pnpm docs:preview
```
