2020 年是改變一切的一年。疫情、居家、不確定性……同時也是技術突破的一年。

## 技術上

**Vue 3 正式版**

等了一年多，終於來了。9 月釋出後，我們內部新專案開始用 Vue 3 + Vite。開發體驗比 Vue 2 + Webpack 好很多，特別是啟動速度。

Composition API 用熟之後，真的很爽——邏輯複用清晰，TypeScript 支援好。但生態還不完整，Element Plus 還在 Beta，用了一些社群元件庫踩了不少坑。

**Vite 1.0**

12 月釋出，決定在下個專案全面採用。告別 60 秒啟動。

**遠端工作適應**

三月開始居家，到現在已經 9 個月了。逐漸接受並優化了遠端工作方式：

- 非同步溝通為主，減少無效會議
- 文件先行，程式碼 Review 更認真
- 專注時間反而比在公司更長

**深入瞭解的技術**

- Vue 3 Composition API 原始碼
- React Concurrent Mode（還在等正式版）
- Vite 的 ESM 原理
- 前端監控系統（自建了一套簡單的）

## 2020 年沒完成的

- 演算法刷題：計劃每週三題，實際平均每月五題（😅）
- 開源貢獻：提了幾個 issue，但沒有實質性 PR
- 技術分享：團隊內分享 3 次，計劃的 6 次

## 影響最深的一件事

被迫全面轉向非同步遠端工作，讓我意識到：

**寫作能力對工程師很重要。**

當面溝通成本低，可以用語氣和肢體語言補充資訊。文字溝通沒有這些，必須把事情說清楚。這一年我寫了更多技術文件、設計文件，溝通質量反而提升了。

## 2021 年計劃

技術：

- 深入 React 18 新特性（Concurrent Mode 應該正式了）
- Vite 2 生產實踐
- 系統學習演算法（認真的那種，每天 1 題）
- 開始看英文技術書（SICP）

工程：

- 搭建團隊前端監控系統
- 推動元件庫建設

個人：

- 部落格每月至少 4 篇（不斷更了）
- 鍛鍊身體（居家久坐太糟糕了）

2020 在混亂中度過，但混亂也是成長的土壤。2021 加油。
